# CE-ESY-assignments
 (Struct): يحتوي على المصفوفة ومؤشرات القراءة (head) والكتابة (tail) بالإضافة إلى عداد العناصر (count)
​دالة التهيئة (init): لضبط المؤشرات والعداد عند بداية البرنامج
​دالة الإدخال (write): لإضافة الأحرف مع التحقق من حالة الامتلاء (Overflow)
​دالة القراءة (read): لاستخراج الأحرف مع التحقق من حالة الفراغ (Underflow)
يستقبل البرنامج اسماً من المستخدم عبر الإدخال القياسي
​يتم دمج السلسلة النصية CE-ESY مع الاسم المدخل
​يقوم البرنامج بتخزين كل حرف من السلسلة الناتجة داخل المخزن الدائري ويتم قراءة البيانات من المخزن وعرضها على الشاشة للتأكد من إفراغه تماماً
